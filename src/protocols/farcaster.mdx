## What is Farcaster?

Farcaster is a sufficiently decentralized social network built on the Ethereum blockchain. It provides users with the ability to create profiles, post short messages or 'casts,' follow other users, and organize into communities. In design, it resembles public social networks like Twitter and Reddit.

## Key Concepts on Farcaster

Accounts: Represent users on Farcaster.

Usernames: Human-readable names associated with accounts.

Messages: Public interactions between accounts.

Apps: Software facilitating account creation, username acquisition, and message posting.

## Farcaster Features

Decentralization: Farcaster is public and decentralized, allowing anyone to build an app to read and write data on the network.

User Ownership: Users own their accounts and relationships, providing the freedom to switch between different apps.
